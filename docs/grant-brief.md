# Grant Brief

## Project

Base Attribution OS is open-source attribution infrastructure for Base builders.
It helps teams add, validate, and enforce Builder Codes across SDKs, wallets,
agents, x402 payment paths, and CI.

## Why Base should care

Base already treats attribution as a key part of rewards, analytics, growth,
and app visibility. This project makes attribution easier to adopt and harder to
break.

## MVP deliverables

- TypeScript core package for ERC-8021 suffix encode/decode/validate.
- viem, wagmi, and ethers adapters.
- `bao` CLI for calldata, transaction, and repo validation.
- GitHub Action for PR enforcement.
- Examples for dApps, wallets, agent flows, and x402 buyer/seller paths.
- Launch docs that help teams ship and share measurable progress.

## Metrics

- GitHub stars and forks.
- External integrations.
- Transaction examples validated by `bao`.
- Base app teams using attribution checks in CI.
- Public case studies and weekly X updates.

## Grant ask shape

The first grant request should be milestone-based:

- Milestone 1: MVP packages and docs.
- Milestone 2: three pilot integrations.
- Milestone 3: Dune query templates and public attribution replay report.
