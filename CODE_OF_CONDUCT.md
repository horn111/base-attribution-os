# Code of Conduct

Base Attribution OS follows the Contributor Covenant spirit: be respectful,
direct, and helpful. This project exists to make onchain attribution easier for
builders, wallets, agents, and ecosystem teams.

Unacceptable behavior includes harassment, discriminatory language, spam,
threats, and repeated bad-faith participation.

To report a problem, open a private channel with the maintainers through
GitHub or contact the project owner at https://github.com/horn111.
